Operating System

Article header with background image and parallax effect

All modern computers have some form of operating system which users generally take for granted. The operating system makes it possible to communicate with the software and hardware that make up a typical computer system. The OPERATING SYSTEM (OS) is essentially software running in the background of a computer system. It manages many of the basic functions shown in the diagram. 

PreviousNext

One of the most common examples if an operating system is known as Windows and is used on many personal computers, other example include: Linux, Android, UNIX and DOS. Windows is an example of a single – user multitasking operating system – this means only one user can the computer at a time but can have many applications open simultaneously. When a computer is first powerup, the initiating programs are loaded into memory from the ROM (Read Only Memory) chip, these programs run a checking procedure to make sure the hardware, processor, internal memory and BISO (basic input output system) are all functioning correctly, if there is no error detected, then the operating system is loaded into memory.
