Internal memory or primary memory

Article header with background image and parallax effect

a. Primary memory features both ROM and RAM as it is essential to store program data and cliché data 

b. Random Access Memory (RAM) is a volatile/temporary memory as the data stored will be loss if the system is powered down. RAM is offered in 2 main types of RAM, Dynamic RAM (DRAM) and Static RAM (SRAM).

c. Read Only Memory (ROM) is nonvolatile/permanent memories as the data will remain even when the system is powered down.

d. RAM and ROM are used for different applications, RAM the user may wish to program in their own routines; these new routines will be stored in RAM chip, while ROM stores factory settings such as firmware update instructions, it also stores the ‘start up’ routines.

PCIE cards
(peripheral component interconnect express)

Article header with background image and parallax effect

a. PCIE cards or also known as specialized cards, these cards help the system in many different ways as these types of card is universal.

b. A wireless network card is an example of specialized card for the example without a wireless network card there will be no Bluetooth or Wi-Fi option on the machine. A Graphic Processing Unit GPU is another example of specialized card where it will bring up the system’s performance.
