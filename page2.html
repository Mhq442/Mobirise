Processor: -


a. Processor is a type of hardware in the computer system, processor often known as CPU (Central Processing Unit) / ‘the brain’. It is a logic circuitry that responds to and process basic instructions that drives the computer. 


b. It does execute all events this means that without a processor the computer system will only show low level languages.


c. As a user you can choose between Intel processor or AMD
